# 🌍 EcoPulse - Carbon Footprint Awareness Platform

## 🎯 Objective
Helps users track and reduce their daily carbon footprint through simple inputs and personalized insights.

## 🚀 Features
- Carbon footprint calculator
- Category-wise CO2 breakdown
- Personalized eco suggestions
- Gamification (eco points & badges)
- Local storage support

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 💡 Approach
User inputs daily activities → system calculates CO2 → displays insights → suggests improvements.

## 📌 Assumptions
- Average CO2 values are used for calculations
- User inputs are approximate estimates

## 🌱 Goal
To increase awareness about climate impact in a simple and interactive way.
